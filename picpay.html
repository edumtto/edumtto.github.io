PicPay

Company site (Portuguese) English page

For over four years, I played a pivotal role in developing various features for PicPay, a widely popular digital wallet app with millions of users. Please note that the app is currently only available in the Brazilian App Store.

Throughout my tenure, I leveraged the following technologies to bring these features to life:

  • UIKit in conjunction with programmatic layout for designing visually appealing and intuitive interfaces.
  • MVVM, Coordinator, and Modular architecture for organizing code and promoting flexibility and scalability.
  • Keychain for securely storing sensitive data.
  • Push notifications for real-time updates and engagement.
  • Camera for scanning QR Codes and Bar Codes, providing seamless and convenient transactions.
  • Dispatch queues for handling asynchronous flow and ensuring optimal performance.
  • REST API connections to facilitate communication with external services.
  • Analytics tools for monitoring usage and gathering insights.
