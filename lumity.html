Project Overview

Lumity Media seeks to enhance user engagement by providing personalized insights based on content consumption. I developed the "Lumity Wrapped" feature, which summarizes activities across books, podcasts, and videos.

The project involved building the data aggregation engine, the visual frontend experience, and the native mobile integration to ensure a cohesive user journey.

Key Responsibilities

  • UI/UX Design in Figma
  • Full-stack Web Development
  • Native iOS Implementation
  • RESTful API Design

Web Implementation

Built with React.js and Vite for speed, styled with Tailwind CSS for a modern, responsive feel. The app consumes the recap API to generate dynamic charts and summaries.

[Screenshot of the React Web Application]

Backend & API

Developed a Node.js service that handles data aggregation. Two critical endpoints were added to the existing codebase:

  • GET /api/v1/recap-data // Returns processed user metrics
  • GET /year-recap // Serves the React frontend bundle

iOS Mobile Integration

Using Swift and UIKit, I integrated the feature into the native app, allowing users to access their recap via a seamless web-view transition that feels native to the Lumity environment.
